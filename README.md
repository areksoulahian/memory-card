# Pixel Memory 🎮

A memory card game with a pixel art theme. Click each card once — click the same card twice and it's game over!

[Play it here](https://areksoulahian.github.io/memory-card/).

## Built with

- **React 18** — functional components with hooks
- **Vite** — fast dev server and builds
- **CSS** — pixel art retro aesthetic with `Press Start 2P` font

## How to play

- Click any card to score a point
- Cards shuffle after every click
- Don't click the same card twice
- Click all 12 to win

## Local dev

```
npm install
npm run dev
```
